Szybki start (dev):

```bash
npm install
npm run dev
```

Uwaga: Do pełnej funkcjonalności powiadomień przeglądarkowych wymagane jest nadanie zgody przez użytkownika. Do logowania Google konieczna jest poprawna konfiguracja projektu Firebase.
